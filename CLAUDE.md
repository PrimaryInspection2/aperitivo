# Claude Code context for Aperitivo

This is an architecture documentation repo for Aperitivo — a Strava-integrated 
endurance athlete platform. Code coming later; for now only docs/.

## Working agreements
- Documentation is in `docs/`. Source files are Markdown.
- Architectural decisions follow Michael Nygard's ADR format in `docs/adr/`.
- Naming: lowercase-kebab-case for filenames, ADRs numbered 4-digit zero-padded.
- Always show diffs before applying.
- Always ask before pushing to remote.
- Always ask before deleting files.
